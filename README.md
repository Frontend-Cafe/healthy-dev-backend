## Description

 Api Rest para aplicación web HealthyDev.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode (nodemon)
$ npm run start:dev

# production mode
$ npm run start:prod
```
